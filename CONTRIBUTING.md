# Contributing

Thank you for your interest in DronePlan.

Before contributing, read these rules carefully.

## Repository status

- This repository is public, but it is not open source.
- Source code is protected by `LICENSE` and `NOTICE`.
- No reuse rights are granted by default.

## Pull requests and patches

- Opening an issue or pull request does not grant you any right to reuse this codebase.
- By submitting code, documentation, or other materials, you confirm that you have the right to submit them.
- By submitting a contribution, you grant the repository owner a perpetual, worldwide, irrevocable right to review, use, modify, relicense, and redistribute that contribution as part of this project.
- Maintainers may reject or remove contributions at their discretion.

## Do not submit

- Secrets, credentials, tokens, certificates, or private keys
- Third-party code that you do not have the right to contribute
- Generated artifacts unless explicitly requested
- Machine-specific settings or local tooling state

## Practical guidance

- Keep changes narrow and reviewable
- Prefer documenting current behavior over writing speculative roadmap text
- Preserve the current Windows-first RC 2 workflow focus
